# Shacky Tale

A Simple role-playing game originally written in Pascal for a college project (1989).

See [shacky_python/README.md](shacky_python/README.md) to see how to run the game.

Source code:
* Original Pascal code: [shacky_t/](shacky_t)
* Ported by ChatGPT Codex to Python: [shacky_python/](shacky_python)
* Ported by ChatGPT Codex to Javascript: [shacky_web/](shacky_python)
  * Runs in a browser. See the [shacky_web/README.md](shacky_web/README.md)

Log files:
* Codex-genedated log: [log.md](log.md)
* Codex-genedated transcript, including the prompts [transcript.md](transcript.md)

Concept, coding, design: Yoonsuck Choe
Music: Ahran Lee, Anya Kim
